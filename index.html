What it does

Type "tall athletic woman in a red superhero suit, hero pose" and get a rigged MPFB humanoid in about a minute. Change your mind? Say "make the suit navy" — it re-renders in 15 seconds. Or pick from 12 hand-tuned presets. Or hit random.

Real humanoid anatomy

Powered by MPFB 2 — clean topology, auto-rigged, 163 bones with fingers + toes.

6 pose presets

Hero, action, relaxed, sitting, T-pose, A-pose. IK-driven so hands actually reach targets.

Unity + Unreal FBX export

One click. Weighted, rigged, engine-conventions applied.

Iterative refinement

"Change eye color to green" runs in 15 seconds. Not a full rebuild.

Random + themed generation

Pure Python RNG across the spec axes. Genuinely diverse, not LLM-converging.

Local-only chat UI

Dark-mode browser interface. Bring your own Claude API key. Zero telemetry.

Requirements

  • Blender 4.2 or newer (4.5 LTS recommended)
  • MPFB 2 add-on (free, one-time install — setup in the docs)
  • Python 3.9+
  • Claude API key for natural-language mode — $5 free signup credit covers hundreds of characters
  • Mac, Windows, or Linux

About

Indie tools for Blender and game dev. This plugin grew out of needing a faster way to make NPCs — typing a description beats clicking through 40 sliders. Updates within the 2.x line are free for all buyers.
